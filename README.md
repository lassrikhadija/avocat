# Beaumont Avocats — site démo (modèle pour avocats)

Site vitrine **de démonstration** pour un cabinet d'avocats / cabinet juridique, conçu par **[NEXTIWEB Studio](https://nextiweb.ca)**.

> ⚠️ **Cabinet fictif.** « Beaumont Avocats » n'existe pas : coordonnées, avocate, statistiques, avis et honoraires sont factices. Ce dépôt sert d'échantillon de portfolio et de modèle réutilisable pour les professionnels du droit.

## Contenu

- `index.html` — version française (FR-CA)
- `index-en.html` — version anglaise (EN-CA)
- `sitemap.xml`, `robots.txt`, `llms.txt` — fichiers SEO / IA (standard 2026)

## Caractéristiques

- **Design hi-tech 2030** — blanc cassé · marron · touche bébé bleu
- **Bilingue** FR/EN avec `hreflang` réciproques et `canonical`
- **UX calibrée sur les douleurs du persona** (anxiété, peur des coûts, confidentialité, urgence)
- **Conforme Loi 25** — bandeau de consentement aux témoins, témoins essentiels uniquement, aucun suivi publicitaire tiers
- **Accessibilité** — lien d'évitement, navigation clavier, contrastes conformes
- **Page autonome** — un seul fichier HTML par langue (CSS + JS inline), aucune dépendance
